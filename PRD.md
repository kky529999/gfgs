> **版本**：v0.1

## 1. 背景

- 公司现有约20人，用 Excel + 微信群管理所有运营数据
- 核心痛点：数据互通混乱，查数据要挨个问人
- 目标：业务管理系统

## 2. 客户生命周期（8阶段）

```
现勘 → 设计出图 → 建档通过 → 备案 → 并网资料+网上国网 → 发货 → 并网 → 闭环
```

| 阶段 | 负责部门 | 时效要求 |
|------|---------|---------|
| 现勘 | 技术部 | 录入现勘日期 |
| 设计出图 | 技术部 | 2天内完成，品牌方审核通过 |
| 建档通过 | 综合部 | 录入建档通过日期 |
| 备案 | 综合部 | 现勘单/金融单录完后 **5工作日** 内完成（网上申报）|
| 并网资料+网上国网 | 综合部 | 备案后准备并网材料，提交网上国网 |
| 发货 | 技术部 | 录入发货日期，同时锁定业务提成价格 |
| 并网 | 技术部 | 天合 **43天**，其他品牌 **28天**（从发货日起）|
| 闭环 | 综合部 | 并网结束后 **10日** 内完成 |

**逾期预警**：截止日前7天触发提醒

**品牌截止日计算**：
- 天合：并网截止日 = 发货日 + 43天
- 其他品牌：并网截止日 = 发货日 + 28天

## 3. 提成规则

### 分期发放
- **进场50%**：施工队进场时发放（需提供进场凭证）
- **闭环50%**：闭环完成后发放

### 结算条件
- 完成闭环
- 为客户拍摄满意视频（含初装补贴画面）
- 提交钉钉系统审核
- 业务费员工需开具 **1%增值税专票**

### 业务部额外奖励
- 当月发货户数 ≥ 业务部全职正式员工数 × 3（且正式员工 ≥ 3人）
- 额外奖励业务部 **2000元**

### 技术部现勘/出图奖励
- 技术人员自己出图：2元/块组件
- 外委出图：30元/户

### 综合部自媒体奖励
- 普通视频（30秒+，点赞20+或浏览100+）：5元/条
- 真人出镜视频（40秒+）：50元/条
- 直播宣传（30分钟+，观看30人+）：100元/场

## 4. 薪资结构

```
当月薪资 = 底薪 + 提成奖励
底薪 = 基本工资 + 岗位工资 + 全勤奖 + 工龄工资
发放时间：次月 20日（如遇假期提前或顺延）
```

- 日底薪 = 月应发薪资 ÷ 26
- 小时底薪 = 月应发薪资 ÷ 26 ÷ 8

## 5. 成长基金

### 奖惩规则
| 行为 | 金额 |
|------|------|
| 微信群超过1小时/休息超8小时未回复 | 10元/次 |
| 工作汇报格式不符 | 5元/次 |
| 未发工作日报（未报备） | 10元/次 |
| 迟到/早退1小时内 | 10元/次 |
| 违纪、泄密、争执 | 10-100元/次 |
| 未请假擅自休假/失联旷工 | 200元/天 |
| 未关闭电器设备 | 10元/次 |

### 考核说明
- 同一岗位同一件事，取最高标准奖惩
- 未达标被考核则不对此项工作进行奖励
- 涉及部门奖惩，由总经理/副总和部门主管在绩效会上商定

| 闭环 | 综合部 | 闭环日期 |

### 统一视图设计

在客户列表页面，每个客户显示为一行，8个阶段用横向进度条或时间线展示：

```
客户：张三 | 区域：渭滨区 | 品牌：天合 | 板数：20块
现勘 ✓  设计出图 ✓  建档 ✓  备案 ✓  并网资料 ✓  发货 ✓  并网 →  闭环
```

- ✓ = 已完成（有日期）

## 15. 客户利润计算

每个客户可自动计算利润：

```
利润 = 品牌方结算 - 施工成本 - 二级商分成 - 逾期扣款
```

### 品牌方结算（从配置读取）
= (安装服务费 + 综合补贴 + 渠道提点) × 板数 + 验收奖励

### 施工成本（并网后填写）
= 人工费 + 材料费 + 其他费用

### 二级商分成
= 每块板子费用 × 板数
- 二级商客户才计算，直销客户为0

### 逾期扣款
- 根据品牌政策中的并网考核规则计算

### 月度达量奖励
- 按月配置，不同月份可能不同
- 单独统计，不合并到单客户利润

### 发票管理
- 每客户开一张发票
- 发票金额 = 该客户的品牌方结算金额
- 记录：发票号、开票金额、开票日期

### 流程
1. 并网（录入并网日期）
2. 填写施工成本
3. 品牌打款（约2周，我们要催）
4. 开具发票
